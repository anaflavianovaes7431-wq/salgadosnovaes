Passo 1 – Criar uma conta no GitHub

  1. Vá em github.com.

  2. Clique em Sign up.

  3. Preencha seus dados (nome, email, senha).

  4. Confirme seu e-mail.
    Pronto, agora você tem uma conta GitHub.

Passo 2 – Criar um novo repositório

  1. Faça login no GitHub.

  2. No canto superior direito, clique no + → New repository.

  3. Dê um nome pro seu site, por exemplo: pizzaria-novaes.

  4. Marque a opção Public.

  5. Clique em Create repository.

Passo 3 – Enviar seus arquivos

Você tem duas opções: pelo navegador ou pelo Git. Vou ensinar pelo navegador (mais fácil):

  1. Abra o repositório que você acabou de criar.

  2. Clique em Add file → Upload files.

  3. Arraste seu arquivo index.html (e se tiver imagens, coloque dentro de uma pasta chamada images).

  4. Clique em Commit changes.

Passo 4 – Ativar o GitHub Pages

  1. No seu repositório, clique em Settings → Pages.

  2. Em Build and deployment, selecione Deploy from a branch.

  3. Em Branch, escolha main e clique Save.

  4. Pronto! Agora o GitHub vai gerar um link do seu site.

    • Normalmente algo como:

      arduino
      https://seunome.github.io/pizzaria-novaes/

Passo 5 – Testar seu site

  • Copie o link e abra no navegador.

  • Seu site vai estar no ar, com imagens, slider e botão de WhatsApp funcionando.

💡 Dicas extras:

  • Se quiser mudar alguma coisa, é só alterar o arquivo index.html e dar Commit changes de novo.

  • Todas as alterações são atualizadas automaticamente no link do GitHub Pages.

  • Para organizar imagens, crie uma pasta images e mude os caminhos no HTML, por exemplo:

    html
    <img src="images/margherita.jpg" alt="Pizza Margherita">
